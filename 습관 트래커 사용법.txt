================================================================
           습관 트래커 사용법 가이드
================================================================

[ 목차 ]
1. 시작하기 - 로그인
2. 습관 추가하기
3. 습관 완료 체크하기
4. 습관 수정/삭제하기
5. 알림 설정하기
6.(강추) 모바일에서 앱처럼 사용하기


----------------------------------------------------------------

1. 시작하기
----------------------------------------------------------------

[ 로그인 ]
- 이메일 로그인 : 이메일과 비밀번호 입력 후 로그인
- Google 로그인 : "Google로 계속하기" 버튼 클릭

* 한 번 로그인하면 다음에 앱을 열 때 자동으로 로그인됩니다.

----------------------------------------------------------------

2. 습관 추가하기
----------------------------------------------------------------

1. 메인 화면 우측 상단 "+ 추가" 버튼 클릭
2. 습관 이름 입력 (예: 물 2L 마시기, 30분 운동, 독서)
3. 알림 시간 설정 (선택 사항) - 알림을 받고 싶은 시간 선택
4. "추가하기" 버튼 클릭

----------------------------------------------------------------

3. 습관 완료 체크하기
----------------------------------------------------------------

- 습관 항목 왼쪽의 원형 버튼을 탭/클릭하면 완료 처리됩니다
- 완료된 항목은 초록색으로 표시되고 취소선이 생깁니다
- 상단 진행률 바에서 오늘의 달성률을 확인할 수 있습니다
- 완료한 항목을 다시 탭하면 완료 취소가 됩니다

* 완료 여부는 매일 자정에 자동으로 초기화됩니다.

----------------------------------------------------------------

4. 습관 수정/삭제하기
----------------------------------------------------------------

1. 수정/삭제할 습관 항목 우측의 ⋮ (점 세 개) 버튼 클릭
2. 메뉴에서 선택:
   - 수정 : 습관 이름 또는 알림 시간 변경 후 저장
   - 삭제 : 해당 습관 완전 삭제

----------------------------------------------------------------

5. 알림 설정하기
----------------------------------------------------------------

[ 알림 권한 허용 ]
- 앱 상단에 알림 허용 배너가 보이면 "허용" 버튼 클릭
- 브라우저에서 알림 권한 팝업이 뜨면 허용 선택

[ 알림 동작 방식 ]
- 습관에 알림 시간을 설정하면 해당 시간에 브라우저 알림이 발송됩니다
- 이미 완료된 습관은 알림이 오지 않습니다
- 앱(브라우저 탭)이 열려 있어야 알림이 동작합니다

[ 기기별 알림 지원 ]
- PC (Chrome, Firefox, Edge)    : 지원
- Android 브라우저               : 지원
- iPhone (홈 화면 추가 후)       : iOS 16.4 이상 지원
- iPhone 일반 Safari             : 미지원

----------------------------------------------------------------

(강추) 6. 모바일에서 앱처럼 사용하기
----------------------------------------------------------------

홈 화면에 추가하면 일반 앱처럼 사용할 수 있습니다.

[ iPhone - Safari ]
1. Safari에서 앱 주소 접속
2. 하단 공유 버튼 (네모에 위쪽 화살표) 탭
3. "홈 화면에 추가" 탭
4. "추가" 탭
5. 홈 화면 아이콘으로 실행

[ Android - Chrome ]
1. Chrome에서 앱 주소 접속
2. 우측 상단 메뉴(점 세 개) 탭
3. "홈 화면에 추가" 탭
4. "추가" 탭

* 홈 화면에 추가하면 주소창 없이 전체 화면으로 사용 가능하고,
  iPhone에서 알림도 활성화됩니다.

----------------------------------------------------------------

7. 자주 묻는 질문
----------------------------------------------------------------

Q. 습관이 추가되지 않아요
A. 인터넷 연결 상태를 확인해 주세요.
   로그인 상태인지 확인해 주세요.

Q. 알림이 오지 않아요
A. 브라우저 알림 권한이 허용되어 있는지 확인하세요.
   앱 탭이 열려 있어야 알림이 동작합니다.
   iPhone은 홈 화면에 추가 후 사용하세요.

Q. 다른 기기에서도 같은 습관 목록이 보이나요?
A. 네, 같은 계정으로 로그인하면 모든 기기에서 동기화됩니다.

Q. 완료한 습관이 다음 날 초기화가 안 돼요
A. 자정(00:00)에 자동 초기화됩니다.
   앱을 새로고침하면 바로 반영됩니다.
